## 📊 仪表板包含 7 个分析部分

### 1. 📊 **Overview** - 总体概览 (最常用!)
- **关键指标**：客户数、贷款率、平均收入、消费
- **快速洞察**：最重要的发现总结
- **对比统计**：过滤前后的数据对比

**最适合**: 主管、领导看5分钟快速了解情况

---

### 2. 📈 **Income Analysis** - 收入分析
- **分布直方图**：按贷款状态的收入分布
- **箱线图**：收入范围和异常值
- **统计对比**：贷款客户 vs 非贷款客户的收入
- **转化率**：不同收入区间的转化率

**发现**: 
- 收入 $150k+ 的客户中 20%+ 会接受贷款
- 收入 < $80k 的客户中只有 < 5% 会接受贷款

---

### 3. 💳 **Credit Card Analysis** - 信用卡消费分析
- **消费分布**：月均消费的分布
- **收入 vs 消费**：散点图展示关系
- **VIP 检测**：异常值识别
- **VIP 统计**：高消费客户的转化率 (32.7%!)

**发现**:
- 消费 $5k+/月 的客户有 32.7% 转化率 (3.4倍!)
- 高收入 + 高消费 = 理想客户

---

### 4. 🎓 **Education Analysis** - 教育程度分析
- **按教育水平的转化率**：本科 vs 研究生 vs 专业
- **教育水平的收入分布**
- **详细统计**：各水平的客户数和特征

**发现**:
- 本科：4.4% 转化率
- 研究生：13% 转化率 (3倍提升!)
- 专业人士：13.7% 转化率

---

### 5. 🎪 **VIP Segment** - VIP 细分
- **Tier 1 VIP**: 150k+ 收入、4k+ 消费、高学历
- **Tier 2 Core**: 100-150k、2-4k 消费
- **High Spenders**: 5k+ 消费的豪客
- **可视化对比**：大小和转化率

**建议**:
- Tier 1: 专业客户经理一对一服务
- Tier 2: 邮件和电话营销
- High Spenders: 高端产品和优先审批

---

### 6. 🎯 **Customer Tiers** - 3 层分层模型
- **分层逻辑**：清晰的分类规则
- **人数分布**：派表图显示大小
- **转化率对比**：柱状图对比
- **行动建议**：每层的具体营销策略

**3 层模型**:
```
Tier 1 (VIP):       150+ / 4k+ / 高学历      → 35-40% 转化
Tier 2 (Core):      100-150 / 2-4k / 高学历  → 18-22% 转化
Tier 3 (Secondary): 80-120 / 1-3k / 任何     → 8-12% 转化
Do Not Pursue:      <80 或 <1k 消费          → < 5% (放弃)
```

---

### 7. 📋 **Data Explorer** - 数据浏览器
- **原始数据**：查看完整数据表
- **统计汇总**：数值统计和对比
- **相关性矩阵**：变量间的相关性热力图
- **下载功能**：导出过滤后的 CSV 数据

## 🛠️ 常见操作

### 如何重新加载数据？
```
1. 刷新浏览器 (F5)
2. 或重启 Streamlit 应用
```

### 如何导出数据？
```
1. 设置过滤条件
2. 进入 📋 Data Explorer
3. 在 "📊 Raw Data" 标签页
4. 点击 "Download filtered data as CSV"
```

### 如何使用数据进行进一步分析？
```
1. 导出 CSV 数据
2. 在 Excel 中打开
3. 使用 Excel 的数据透视表
4. 或导入到 Python/R 进行分析
```

### 如何生成报告？
```
1. 在浏览器中使用 Ctrl+P (打印)
2. 选择 "保存为 PDF"
3. 保存报告
```
